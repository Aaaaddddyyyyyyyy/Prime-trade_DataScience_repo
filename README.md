## 🚀 Overview

This project performs an **end-to-end data science analysis** to understand how **Bitcoin market sentiment (Fear vs Greed)** influences:

- 📈 Trader performance (PnL, win rate)  
- ⚡ Risk behavior (leverage usage)  
- 🔁 Trading patterns (frequency, position bias)  

💡 The goal is to convert raw trading data into **actionable strategy insights**.

---

## 🧠 Problem Statement

> Can market sentiment predict trader success, behavior, and risk-taking patterns?
